Use Hexo To Create Blog

Veröffentlicht am 2019-07-06

环境搭建

下载node

访问https://nodejs.org/zh-cn/
下载一个版本，然后根据步骤安装

安装hexo脚手架

（window系统）打开dos命令窗口
输入命令 npm install [-g] hexo-cli 如果选择全局安装就使用-g命令，如果不使用就去掉，推荐使用全局安装

安装淘宝镜像

如果使用npm命令安装插件时，发现下载速度过慢，或者反复安装失败的，可以下载淘宝镜像
在命令窗口输入 npm install -g cnpm –registry=https://registry.npm.taobao.org

建立工程文件

新建一个文件夹，然后进入文件夹，在命令窗口相应的文件路径里输入命令
hexo init 然后工具会自动创建一些文件在里面
输入 hexo s 可以本地访问，打开浏览器输入 localhost:4000
我们就可以看到hexo自动给我们创建的博客

新建文章

在source/_posts文件夹里面建立一个后缀名为md的文件
或者是在命令窗口输入 hexo n “文件名”
内容可以自己写，规则网上可以百度查看
新建立文章后我们必须输入 hexo g 生成相应的文章页面

更改页面样式

觉得页面不好看，或者修改信息时，我们进入public文件里面，根据自己的要求修改index.html页面
和添加自己编写的js文件和css样式
