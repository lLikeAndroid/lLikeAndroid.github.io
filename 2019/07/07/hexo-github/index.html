Use Github Deploy My Hexo Blog

发表于 2019-07-07

1.建立github账号

前往 https://github.com/ ，申请一个账号

2.建立仓库

在github个人页面里点击右上角加号，选择new repository
然后填写repository名字，必须严格按照格式填写 注册账号的名字.github.io
然后创建仓库

3.安装部署插件

在命令窗口输入 [cnpm]npm install –save hexo-deployer-git

4.修改配置文件

打开工程文件里面_config.yml文件，然后找到deploy字段，修改成
deploy:
type: git
repo: 新建立的仓库的http地址
branch: master
注意冒号后面必须接一个空格

5.部署到github

在命令窗口输入 hexo d
